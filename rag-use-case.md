# Use Case 1: HR Assistant

## Problem

Employees frequently ask questions about company policies.

## Documents

* Leave Policy PDF
* Attendance Policy PDF
* Employee Handbook

## User Questions

* How many casual leaves do I get?
* What is the work-from-home policy?
* What is the notice period?

## Why RAG?

ChatGPT does not know the company's internal policies. A RAG system retrieves answers directly from HR documents and provides accurate responses.

# Use Case 2: Course Notes Assistant

## Problem

Students spend time searching through lecture notes and PDFs.

## Documents

* Lecture Notes
* Course PDFs
* Assignments

## User Questions

* Explain Chapter 5.
* Summarize Unit 3.
* What are the key concepts of Module 2?

## Why RAG?

The assistant retrieves information from the uploaded study materials instead of generating generic answers.

# Use Case 3: Product Manual Assistant

## Problem

Customers need help using products but do not want to read long manuals.

## Documents

* Product User Manual
* Installation Guide
* Troubleshooting Guide

## User Questions

* How do I reset the device?
* How do I update the firmware?
* What does error code 404 mean?

## Why RAG?

The assistant retrieves the exact instructions from the manuals and provides accurate guidance.
